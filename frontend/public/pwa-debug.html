Advanced Debugging

Chrome DevTools Check:

  1. Open DevTools (F12)
  2. Go to Application tab → Manifest
  3. Look for "Installability" section
  4. If it shows errors, they will be listed there
  5. If it shows "No issues", the PWA is installable!

Manual Install (if beforeinstallprompt doesn't fire):

  • Chrome/Edge Desktop: Menu (⋮) → "Install Graditone..."
  • Chrome Android: Menu (⋮) → "Add to Home screen" or "Install app"
  • Safari iOS: Share button → "Add to Home Screen"
