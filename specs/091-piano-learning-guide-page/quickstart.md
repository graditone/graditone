# Quickstart: Piano Learning Guide Page — Feature 091

## Prerequisites

- Node 22 (managed via `.nvmrc`)
- Run `npm install` in `frontend/` before starting

```bash
# Start the dev server
cd frontend && npm run dev
```

## Verify the guide page

1. Open http://localhost:5173 in your browser
2. Navigate to the guide or help section from the landing screen
3. Confirm the "Piano Learning" page is listed and loads correctly
4. Verify all sections are visible: overview, workflow, piano features, tips

## Run tests

```bash
cd frontend && npm test
```
